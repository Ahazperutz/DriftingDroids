DriftingDroids - yet another Ricochet Robots solver program.

version: (work in progress 2011-09-17)



// about

the original board game by Alex Randolph:
http://en.wikipedia.org/wiki/Ricochet_Robot
http://de.wikipedia.org/wiki/Rasende_Roboter
http://boardgamegeek.com/boardgame/51/ricochet-robots



// usage

this program is still work in progress. (see TODO.txt)
and this documentation is incomplete, too...

Java SE Runtime Environment (JRE version 6 or later) is required to this program.
Download here:
http://www.oracle.com/technetwork/java/javase/downloads/index.html

Windows users please rename the file "start_cmd.txt" to "start.cmd" and
doubleclick "start.cmd" to run it.
